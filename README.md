# 🎾 Tennis Match Prediction using Machine Learning

This project predicts the outcome of tennis matches using machine learning.

## 🚀 Features Used
- Player Rankings
- Rank Difference
- Rank Ratio
- Surface Type

## 🤖 Model
- Random Forest Classifier

## 📊 Accuracy
~60–70%

## 📂 Dataset
ATP Tennis Matches Dataset

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the model:
python model.py

## 🧠 Key Learnings
- Data preprocessing
- Feature engineering
- Model training and evaluation
